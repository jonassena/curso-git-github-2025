git init .           -> Cria um repo git na pasta atual
git status           -> Mostra o status do trabalho
git add nome_arquivo -> Adiciona arquivo para ser comitado
git commit -m "msg"  -> Faz commit do que estava para ser comitado
git log              -> Mostra todos o histórico de commit
git reset            -> Remove o arquivo do stage
git branch           -> Mostra as branchs que existem e indica a branch de trabalho
git branch -b "nome" -> Cria uma nova branch
git checkout         -> Muda de branch
git merge            -> Mescla as branchs


## Fluxo de Trabalho
1 - Criar o diretório (pasta) do projeto
2 - Iniciar o git dentro do diretório criado (git init .)
3 - Edita ou criar arquivos
4 - git status
5 - git add nome arquivo
6 - git status
7 - git commit -m "mensagem"
8 - git status
9 - Volta para o passo 3
